Tags de Formatacao de Texto

Este e um texto de exemplo para demonstrar as tags de formatacao do HTML.
Podemos deixar palavras em negrito forte quando queremos dar enfase.
Tambem podemos usar negrito simples para destacar termos importantes.
O HTML permite escrever em italico para dar um estilo diferente ao texto.
Este texto esta maior que o normal usando a tag big.
Podemos escrever texto sobrescrito como em formulas: x2 + y2.
E tambem texto subscrito como em formulas quimicas: H2O.
Texto com riscado usando a tag s indica algo removido.
Texto com riscado usando a tag strike tambem indica remocao.
Podemos sublinhar palavras importantes para chamar a atencao do leitor.
